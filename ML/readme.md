# 3. AUTOML
- 평가지표
    - 정확도(Accuracy) 일반적 성능에 대한 평가지표    
    = $ (TP + TN) / (TP + FN + FP + TN) $    
       
         
    |구분|예측: 거짓|예측: 참|
    |--|:--:|:--:|
    |실제: 거짓|TN|FP|
    |실제: 참|FN|TP|

    - 정밀도(Precision) : 모델이 True라고 예측한 것 중에서 실제 True인 것의 비율   
    = $ TP / (FP+TP) $ 
    - 재현율(Recall) : 실제 True인 것 중에서 모델이 True라고 예측한 것의 비율    
    = $ TP / (FN+TP) $
    - 정밀도와 재현율은 트레이드 오프관계에 있다.
        - $ F1 Score = 2 * (정밀도 * 재현율) / (정밀도 + 재현율) $
    - ROC_AUC커브 -> ROC_AUC_SCORE
        - 모델의 적합성 평가
